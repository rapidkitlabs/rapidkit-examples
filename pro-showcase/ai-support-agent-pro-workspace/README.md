# AI Support Agent Pro

AI customer support automation workspace planned with RapidKit + Workspai.

This is the public showcase page for a private Pro workspace. The full implementation, release evidence, customer archive, entitlement logic, and buyer documentation stay in the private `rapidkit-examples-pro` repository until the product reaches release-candidate status.

## Who It Is For

Teams building support automation systems that need a faster path to:

- AI assistant APIs and provider routing
- Agent runtime, RAG, prompt operations, and guardrails
- Customer support ticket workflows
- Background jobs, notifications, and connector orchestration
- Admin controls, feature flags, tenancy, and audit policy
- Workspace-level verification before customer delivery

## Free vs Pro

| Area | Free AI Agent Workspace | AI Support Agent Pro |
| --- | --- | --- |
| Source access | Public example workspace | Private customer package when approved |
| Scope | Learning/reference AI assistant scaffold | Multi-service support automation workspace |
| Services | FastAPI and NestJS AI assistant examples | API, AI runtime, worker, admin, support workflows |
| Evidence | Public docs and source | Planned release gates, package evidence, and buyer docs |
| Best for | Evaluation and learning | Commercial support automation delivery |

## Start Free

Use the public AI Agent Workspace first:

```bash
git clone https://github.com/rapidkitlabs/rapidkit-examples.git
cd rapidkit-examples/my-ai-workspace
npx rapidkit doctor --workspace
```
